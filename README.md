# ServerClient
Консольное приложенние "Клиент-Сервер" работает следущим образом: \n
1-Запускаем сервер;
2-Запускаем клиента;
3-Вводим серверу сообщения, которые нам возвращаются эхом;
4-Для выхода из чата требуется написать три ключевых слова "exit","close","выход".
